Register

Title: Register

Actor: Any potential user of the app.

Prerequisite: The user should think of a unique username and secure password.

Action: The user should type in their username and password.

Effect: The user becomes registered and has their user name and password. Unless, if the username they want to use is already registered, the user is given a message explaining the issue and returned to the register page.

Next: The user is sent to the home page to login using their newly created username and password.
