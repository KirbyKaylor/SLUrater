Title: Admin Delete

Actor: A registered and logged in user and admin of the app.

Prerequisite: The user is logged in and is an admin. Another user of the app has flagged a comment.

Action: The user chooses to delete an inappropriate comment.

Effect: The user receives a confirmation message, and once confirmed, the comment is deleted.

Next: None, or the user retuns to the home page.
