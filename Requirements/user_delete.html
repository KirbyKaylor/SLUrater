UserDeleteComment

Title: UserDeleteComment

Actor: A user.

Prerequisite: The user must be logged in, and must have written the comment they wish to delete.

Action: The user clicks the delete button by the comment they wish to delete.

Effect: The comment is deleted from the site and its database.

Next: The page is refreshed and the comment is no longer displayed.
