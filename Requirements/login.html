Login

Title: Login

Actor: A user or administrator.

Prerequisite: The user must have a registered username and password.

Action: The user or administrator must type in their username and password and click the login button.

Effect: The user is either successfully logged in or an error message is given.

Next: If successful, the user is sent to the home page. If unsuccessful, there is no action; the user remains on the login page.
