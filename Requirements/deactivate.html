Deactivate Account

Title: Deactivate Account

Actor: Any potential user of the app.

Prerequisite: The user must be logged in and currently at their home page.

Action: The user clicks a button 'Deactivate Account'

Effect: Any information related to the user in the database is created. A new user can now use the username of the account just deactivated.

Next: The User is sent back to the home page of the app.
