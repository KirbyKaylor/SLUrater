Navigation

Title: Navigation

Actor: Any user of the app.

Prerequisite: None.

Action: The user clicks a link to a category in the navigation menu.

Effect: The user reaches the page for the category they clicked on.

Next: The user may select a subcategory to which they would like to navigate.Otherwise, they may create a new review within the category
