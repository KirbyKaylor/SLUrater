External Links

Title: External Links

Actor: Any potential user of the app.

Prerequisite: The user must be in a category searching for a page or at an event page

Action: The User clicks the external link of the category or of an event page

Effect: A new tab is opened with the link of the external in the category or the event page

Next: The user is sent to the next tab so they can review the information on the page about the category or the event on the event page
