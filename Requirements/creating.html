User Creating an Event

Title: User Creating an Event

Actor: Any potenital user of the app

Prerequisite: The user must be logged in in order to create an event and the event must already fall under a category on the app.

Effect: When a user enters in the information in the "Create an event" form, a new event is then entered into the databse and hence the web app.

Next: The user is sent to the event page to potentially rate the new event or write a comment.
