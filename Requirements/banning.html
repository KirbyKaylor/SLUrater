Banning a User

Title: Banning a User

Actor: An administrator of the app

Prerequisite: The administrator believes a user is being detrimental to the website and its users because of comments left or because of other unwanted behavior

Action: An administrator clicks the "Ban User" option when viewing all the accounts with the app

Effect: Any information relating to the user in the database is deleted. A new user can now use the username of the account just banned.

Next: The administrator will no longer see that account in the account listed with the app.
